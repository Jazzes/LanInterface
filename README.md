# Interface

### 1. Склонируйте репозиторий:

```
git clone https://github.com/Jazzes/LanInterface.git
```

### 2. Перейдите в папку:

```
cd LanInterface
```


### 3. Подключите библиотек:

```
npm i
```

### 4. Запустите проект:

```
npm start
```

### 5. Для сборки проекта используйте:

```
npm run build
```
